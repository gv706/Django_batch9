Project Creation:
================
- In Particular folder : open CMD: django-admin startproject DemoProject(Project Name)
- cd DemoProject 
- RUN SERVER : python manage.py runserver portno 

- Project Folder:
	- __init__ : Consider a Python File(Package)
	- settings.py : All App info and DataBases
	- urls.py : All URLs using browser related
	- wgsi : webserver gateway interface
